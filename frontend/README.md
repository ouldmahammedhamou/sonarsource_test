# Frontend Angular — Personnages Wiki

## Lancer en dev

1. Installer les dépendances : `npm install`
2. Démarrer le backend Spring Boot (port 8080)
3. Démarrer Angular : `npm start` → http://localhost:4200

## Pages

- **/list** — Liste des personnages (lien sur le nom → détail)
- **/create** — Formulaire de création (stats 6, 8, 10, 12)
- **/perso/:id** — Fiche détail d’un personnage

## Build

`npm run build` — sortie dans `dist/personnage-wiki-client`.
